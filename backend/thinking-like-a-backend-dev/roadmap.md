### Web services vs API
API are the set of definitions and protocols through which software components work. This can be the app on your phone using the Camera API, but also some RESTful API to communicate with the server. Thus, a web service is a type of API. 

Historically, web services refer to the old SOAP protocol which was extremely verbose in comparison to modern REST api and jsonRPC, utilizing xml in obligatory envelope, head, body elements just for a simple data like the userId. For some reason, old devs use web servcies to refer to SOAP, while API is shorthand for RESTful API.
